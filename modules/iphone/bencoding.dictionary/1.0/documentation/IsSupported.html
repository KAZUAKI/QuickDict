benCoding.Dictionary Module

Method: isSupported

Description

The isSupported method is used to determine if the device has the dictionary service available.  Please note Apple introduced this feature with iOS 5 so this feature will not work on devices running older versions of iOS.

For a detailed example please see the app.js included in the example folder.

Accessing the benCoding.Dictionary Module

Before you can start using any of the module's functionality you need to use the require keyword to make the module accessible from JavaScript. To access this module from JavaScript, you would do the following:

var referenceLibrary = require('bencoding.dictionary').createReferenceLibrary();

Using isSupported

The below example shows how to create the ReferenceLibrary object and check if this feature is supported.

Ti.API.info("This feature is only supported in iOS5 and above");
Ti.API.info("Check if we have the min OS version needed");
var featureSupported = referenceLibrary.isSupported();
Ti.API.info("Is Supported? => " + featureSupported);

 

Why do I use this method?

You will want to check this method before trying to use of the module's features.  Please review the documentation for each method on what happens if the method is called but the device does not support the feature.
